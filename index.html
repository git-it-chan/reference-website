HTML REGIONS

<head> is an element found at the top of the document that contains the title of the page, styles, meta-tags, and scripts that are to be used within said document.

<header> is an element that contains introductory content.

The <nav> element defines the navigation links found on the site.

As its name suggests, the <main> element defines the main content of the page.

The <aside> element definies content that is aside from the main content, like the oldschool web pages found back in the day.

The <footer> element defines the footer. Typically, it can include, copyright information, contact information, a sitemap, or links relating to the document.

The <section> tag groups content that is related

<article> is a tag that is standalone content, like a blog post, or newspaper article.

The <p> tag defines a paragraph.

The <strong> tag is used to define text that is of strong importance, it should not be used to only format bold text.

The <b> tag is used to specify bold text, and should be used instead of <strong>, if the text is not extra important.

The <div> element is used for styling purposes with css, also all the following elements within it are effected by whatever of its styling.

The tags for listing items are as follows:

  • The <ul> tag is used when creating an unordered list.
  • The <li> tag is used for every line within the list.
  • When a list is being coded that needs to be in order, the <ol> tag is used instead.
